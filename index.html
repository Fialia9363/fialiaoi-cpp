强大的特色功能

中文化界面

完整的本地化支持，菜单、按钮、状态栏、提示信息全部中文化，提供更好的用户体验。

智能代码编辑

代码高亮、自动缩进、代码补全、函数签名提示，让编程更加高效便捷。

文件树管理

直观的文件树界面，支持打开文件夹、双击打开文件，方便项目文件管理。

多标签编辑器

支持多文件同时编辑，标签页管理，轻松在不同文件间切换。

多终端支持

可新建多个终端，支持shell命令执行，每个终端可独立运行和中断子进程。

可配置环境

完整的配置界面，支持主题切换、字体设置、编译器路径配置等，配置持久化保存。

关于我们

我们的使命

致力为所有广大Oler创造一个可以使用一个软件编译所有语言的工具。

起源

Fialiaoi-cpp起源于一名初中生Oler对信息学和软件开发的热爱，在使用长久Dev-cpp、Clion、Vs code等编码工具后突发奇想，将这些工具的优点结合在一块儿，打造出了这款软件。

从每一个版本、每一个历程，Fialiaoi-cpp都做出了不一样的改变，不像其他软件变化微乎其微，Fialiaoi现在还在持续维护并开发中，欢迎各位Oler来下载并使用该软件。

Beta1

最新版本
中文化界面（菜单、按钮、状态栏、提示）
文件树（打开文件夹、双击打开）
多标签编辑器（多文件）
代码高亮（pygments；不可用时平滑降级）
自动缩进（回车继承缩进，冒号后增加缩进；Tab → 4 空格）
代码自动补全（Ctrl+Space，基于 jedi；无 jedi 时间退到简单补全）
悬停显示文档/调试时显示变量值（浮动可滚动窗口）
函数签名提示（输入（弹出签名，输入，高亮当前参数）
多终端支持（可新建终端，终端内可执行 shell 命令；每个终端可运行子进程并被 Stop 中断）
运行（Python / C via GCC）输出到终端，支持中断（terminate/kill）
配置界面并持久化到 config.json（GCC 路径、字体、主题、自动保存间隔）
底部状态栏（文件名、光标位置、调试状态文字 + 小圆点、系统时间、内存、CPU）
自动保存、浅色/深色主题切换、行号、简单断点占位（可扩展）
